# Deployment

1. `git clone https://github.com/dima-kov/bookgo.git`
2. `cd bookgo`
3. `mkdir src`
4. `mv -v * ./src`
5. `cd src`
6. `make install_local`

# Front

1. Install Node.JS
2. Install dependencies: `npm install`

## Usage
1. `gulp less` - generate css from less
2. `gulp minify-css` - minify css
3. See more in `gulpfile.js`
